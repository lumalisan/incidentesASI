*************************************************************************
******************* CÓMO IMPLEMENTAR LA APLICACIÓN **********************
*************************************************************************

1. Instalar XAMPP o distribución de Apache similar
2. Crear la base de datos IMPORTANDO los datos mediante el fichero "adminsistemas.sql"
3. Descomprimir "practica.rar"
4. Colocar la carpeta descomprimida en el fichero de "htdocs" creado por XAMPP, yo lo tengo localizado en "D:\xampp\htdocs"
5. Ejecutar la aplicación de XAMPP y encender el Apache y MySQL
6. Ir al navegador y escribir la dirección de ejecución de la aplicación, por defecto es "http://localhost/practica/"
7. Loguearse con el usuario deseado o crear uno nuevo. Algunas credenciales ya creadas (las contraseñas son siempre "1234"):
	a. admin 1234
	b. tecnico1 1234
	c. cliente1 1234
8. Disfrutar de la experiencia de un sistema de tickets hecho en web! :)
